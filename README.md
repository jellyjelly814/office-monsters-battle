## 玩法循环

```
[走位躲怪] → [AI 提效自动旋转攻击] → [打败一个敌人]
                                          │
                            +1 摸鱼值 / +1 XP
                                          │
                  ┌───────────────┬───────────────┐
                  ▼               ▼               ▼
          摸鱼值满 20       XP 满 → 升级       走到咖啡机
          自动 +1 Token     +1 件 AI 工具       每 3s +1 Token
                            或 +20% 转速
                                          │
                                          ▼
                          [连点厕所 5 下 🚽]
                          → 整顿职场模式 10s
                          (无敌 + 转速 ×100)
```

唯一操作:**走位**(手机虚拟摇杆 / PC WASD)。其他全自动,不用按技能、不用选卡。

## 设计支柱

| 我们做 | 我们不做 |
|---|---|
| 60 秒一局,撑满即胜,死亡即败 | 长局、剧情、章节 |
| 1 种武器 + 2 维升级 | 多武器 build / roguelike 选卡 |
| 4 种差异化敌人,无小怪无 boss | 复杂敌人池 / 阶段 boss |
| 摸鱼值 + 咖啡机回血 | 道具掉落 / 商店 / 背包 |
| 厕所彩蛋 | 主流难度调节系统 |
| 8-bit 像素 + 飞书妙搭部署 | 商业化 / 付费 / 体力 |

**核心简化哲学**:能砍的都砍。打开即玩、玩完即关。
