# 职场牛鬼蛇神大作战 — v2.7 游戏设计方案

> **顶部 banner：AI 提效 VS 职场牛鬼蛇神 — 60s，AI 替你把班味打下去！**
> 60 秒办公室生存。横屏俯视，单手摇杆，1 种武器 + 2 维成长，4 种差异化敌人。撑过下班点就是胜利。
> **⚙ 术语等价**：文中 `HP` / `生命` / `回血` / `续命` 与游戏内显示的 `Token` 等价（v2.6 主题词重命名为 Token，10 起递减；公式与数值不变）。

---

## 0. 一句话概念

**「60 秒办公室爆肝割草。AI 提效旋转飞、摸鱼值换 Token，撑到下班就赢。」**

主角是社畜「人人」（**10 Token**），在办公室里走位躲避 4 种**差异化**职场恶魔，靠**AI 提效自动旋转**反击。两条 Token 充电通道：① 每杀 1 怪 +1 摸鱼值，满 20 自动 +1 Token；② 走到茶水间咖啡机旁自动 +Token。撑过 60 秒 → **「今天按时下班！！」**；Token 归零 → **「今晚加班！！！」**。

### Hook 三件套
1. **职场反讽共鸣** — 4 种敌人都是真打工人的 PTSD：绩效面谈 / 画饼大师 / 甩锅同事 / 死催进度
2. **摸鱼值经济** — 杀怪 → 摸鱼 → 咖啡 → Token 充电，正反馈飞轮
3. **厕所彩蛋** — 连点 5 下厕所触发离职信：武器转速 ×100、敌人速度 ×5，秒变摆烂屠杀场

## 1. 设计支柱

| 我们做 | 我们不做 |
|---|---|
| 60 秒一局，撑满即胜，死亡即败 | 长局、剧情、章节 |
| 16:9 横屏俯视 | 竖屏、横版卷轴、3D |
| 手机单手摇杆 / 滑动优先 | 多键位组合 |
| **1 种武器**（AI 提效旋转），**2 维升级**（图标数 + 转速）| 多武器 build、roguelike 选卡 |
| **4 种敌人**，无小怪无 final boss | 复杂敌人池、阶段 boss |
| **摸鱼值（满 20 → +1HP）+ 茶水间咖啡机（路过自动回血）** | 道具掉落、商店、复杂背包系统 |
| 厕所彩蛋（连点 5 下） | 任何主流难度调节系统 |
| 8-bit 像素风 + 飞书妙搭部署 | 商业化、付费 |

**核心简化哲学**：能砍的都砍。游戏只有 1 种武器、4 种敌人、1 个彩蛋、2 个回血来源。

## 2. 核心玩法循环

```
┌──────────────────────────────────────────┐
│         60 秒倒计时开始                  │
▼                                          │
[走位躲怪] ─→ [AI 提效旋转自动攻击] ─→ [打败]   │
                                       │  │
            +1 摸鱼值（满 20 → +1HP）    │
            +1 XP（满级触发升级）         │
                                       │  │
[升级] ─→ +1 件 武器 OR 转速 +20%（交替）  │
                                       │  │
[路过咖啡机] ─→ 自动每 3s +1 HP            │
                                       │  │
[连点厕所 5 下] ─→ 离职信彩蛋触发！       │
                                          │
┌────────────────┐                        │
│ HP ≤ 0?        │ ─ 是 → 「今晚加班！！！」│
│ 60s 到？       │ ─ 是 → 「今天按时下班！！」
└────────────────┘
```

操作：移动（手机摇杆 / PC WASD），其他全自动。**唯一需要点击的是厕所彩蛋按钮**。

## 5. 武器系统：1 种武器 + 2 维成长

### 5.1 唯一武器：AI 提效旋转

| 维度 | 描述 |
|---|---|
| 视觉 | 玩家身边围绕飞舞的 4 个 AI 工具图标徽章（圆形，橙/绿/深蓝/紫四色，对应 `assets/weapon.png`）|
| 攻击模式 | 绕玩家中心做圆周运动，碰到敌人造成 **2 dmg** + 0.3s 击退 |
| 碰撞判定 | 图标与敌人重叠即伤害（每图标对每敌人 0.5s 内最多 1 次伤害，防止瞬秒） |
| 初始数值 | 1 个，旋转半径 80px，转速 1.0 圈/s |
| 图标数上限 | 8 个（均匀分布在圆周上） |
| 转速上限 | 3.0 圈/s |

### 5.2 升级路线（固定交替）

每次升级**自动**应用以下变化（不暂停游戏，不弹卡）：

| 等级 | 触发 | 变化 |
|---|---|---|
| Lv.1 | 开局 | 1 个，1.0x 转速 |
| Lv.2 | XP 达标 | +1 个 → **2 个**，1.0x |
| Lv.3 | XP 达标 | 转速 +20% → 2 个，**1.2x** |
| Lv.4 | XP 达标 | +1 个 → **3 个**，1.2x |
| Lv.5 | XP 达标 | 转速 +20% → 3 个，**1.44x** |
| Lv.6 | XP 达标 | +1 个 → **4 个**，1.44x |
| Lv.7 | XP 达标 | 转速 +20% → 4 个，**1.73x** |
| Lv.8 | XP 达标 | +1 个 → **5 个**，1.73x |
| Lv.9 | XP 达标 | 转速 +20% → 5 个，**2.07x** |
| Lv.10 | XP 达标 | +1 个 → **6 个**，2.07x |
| ...（最高满级 Lv.14：8 个 + 3.0x） |

**XP 曲线**：从 Lv.N 升到 Lv.N+1 需要 `10 + (N-1)×2` 杀（参考 KungfuMaster）。
- Lv.1→2：10 杀；Lv.2→3：12 杀；Lv.5→6：18 杀；Lv.10→11：28 杀
- 60 秒平均打败 ~150-200，预期能升到 **Lv.8-10**

升级瞬间触发**Build 跃迁视觉爆点**：屏幕震动 0.3s + 白色波纹扩散 + 中央飞字 "升职加薪 +1"（保留 Hook 的爽感）。

## 6. 敌人系统：4 种差异化敌人

数值参考 [KungfuMaster](https://github.com/jellyjelly814/KungfuMaster) 的"均衡型 / 高速脆皮 / 高血肉墙 / 最强综合"四档差异化，对应到办公室主题。

| ID | 名字 | 视觉 | HP | 速度 | 接触伤害 | 远程 | XP | 出场 | 定位 |
|---|---|---|---|---|---|---|---|---|---|
| **B1** | 绩效面谈 | 戴金链子的西装大头怪 | **6** | **50** | **1** | — | 1 | 0s+ | 均衡型主力（高密度刷新） |
| **B2** | 画饼大师 | 嘴里飘大饼气泡的笑脸主管 | **8** | **40** | **1** | 画饼弹 1 dmg + 5s 减速 30% | 1 | 10s+ | 远程辅助 |
| **B3** | 遇事甩锅 | 戴帽举牌"不是我责任"的同事 | **10** | **60** | **1** | 锅反射弹 1 dmg | 2 | 20s+ | 高血肉墙（最难磨） |
| **B4** | 死催进度 | 戴 deadline 倒计时帽，红眼狂奔 | **4** | **110** | **3** | — | 1 | 30s+ | 高速突袭（最高单击伤害）|

**差异化设计意图**：
- B1：基础威胁，靠"密度+频率"造成持续压力
- B2：让玩家"被画饼后变慢"，强化反讽 + 节奏破坏
- B3：让玩家"打多了反伤自己"，鼓励走位 + 切换目标
- B4：1 击 3 dmg 是 v2.3 最危险数字（30% HP_MAX），强迫玩家关注高速冲刺方向

### 6.1 反讽台词（出场口头禅 + 对峙吼声 + 死亡反讽）

每个敌人三类台词都做了扩池（详见 `src/balance.js` 的 `SPAWN_TAUNT` / `KILL_TAUNTS` 字段）：

| ID | 出场口头禅（首次刷新时弹大气泡） | 对峙吼声 / 死亡反讽（结算页随机抽 1 句） |
|---|---|---|
| B1 绩效面谈 | "这季度只能给你 B" | 「年底优化名单见」「你态度有问题」「期望值不够」「OKR 完成度 60%」「我们考虑下调一级」「明年 P 序列冻结」 |
| B2 画饼大师 | "再扛半年期权兑现" | 「明年咱们就上市」「上市了你能买套房」「这事关公司未来」「再加一把劲」「年终包够你过年」「期权 1:100 拆分」 |
| B3 遇事甩锅 | "这个不归我管" | 「这事你应该早提一周」「我以为你会做」「之前没人告诉我」「锅不该我背」「你 owner 这事吧」「我这边一直在等你」 |
| B4 死催进度 | "今晚 9 点前必须上线" | 「客户在催了！！！」「明早 review，今晚必须出」「P0 事故，马上来」「现在！立刻！马上！」「我已经回复客户了」「来不及解释了快」 |

### 6.2 玩家打败敌人时（绿色飘字）
「已读不回」、「OOO」、「年假已批」、「下班锁屏」、「自动回复 set」、「祖传不回消息」、「周末是我的」、「请假理由头疼」、「拉黑了」、「AI 已秒回」、「工位无人」、「不在工位」（连杀 30 个触发「📩 离职申请已提交」）

### 6.3 60 秒难度曲线

| 时间段 | 同屏上限 | 刷新率 | 主要威胁 |
|---|---|---|---|
| 0-10s | 8 | 1/s | B1 |
| 10-20s | 16 | 2/s | B1 + B2 |
| 20-30s | 28 | 3/s | B1 + B2 + B3 |
| 30-40s | 40 | 4/s | 全部 4 种混刷 |
| 40-50s | 60 | 5/s | 4 种混刷 + 偶尔双 B4 冲刺 |
| 50-60s | 80 | 6/s | 全员加速（速度 ×1.3）冲终局 |

## 7. 经济系统：摸鱼值 + 茶水间双回血

### 7.1 摸鱼值（主回血通道）

| 规则 | 数值 |
|---|---|
| 每杀 1 怪 | +1 摸鱼值 |
| 累计满 **20** | **自动**触发：+1 HP（已满则浪费），摸鱼值清零重新累计 |
| 视觉反馈 | 满 20 时屏幕弹咖啡杯飞入心心动画 + "干了！" 飘字 + 短促"咕咚"音效 |

预期 60 秒能触发 **7-10 次**回血。

### 7.2 茶水间咖啡机（路过自动回血）

| 规则 | 数值 |
|---|---|
| 玩家进入咖啡机 24px 半径 | **自动**每 3 秒触发 +1 HP（已满则浪费）|
| 玩家离开半径 | 计时器暂停（不持续回血）|
| 视觉反馈 | 站在咖啡机旁时玩家头顶有"☕"小图标轻微旋转 + 每次回血时咖啡杯飞入心心 |

**为什么是"自动"而非"拾取"**：v2.3 简化方向不要道具/背包系统。走过去就回血，纯心流。

### 7.3 双通道平衡

- **摸鱼值**：靠"杀得快"换回血，鼓励主战场搏杀
- **茶水间**：靠"暂时撤退"换回血，鼓励空间走位
- 两者互补，避免单一最优策略
- 没有任何道具掉落、商店、背包概念——保持极简

### 7.4 摸鱼值 vs XP

杀 1 怪同时 +1 摸鱼值 + 1 XP，两个数值独立累计。HUD 上分别显示两条进度条。

## 8. 厕所彩蛋：连点 5 下

### 8.1 触发方式

茶水间区域有一个**厕所图标按钮**（小型门状图标，约 24×24 像素）。玩家**5 秒内连续点击 5 次**触发彩蛋。

### 8.2 触发效果

```
[屏幕震动 1s] + [离职信静态图居中弹出 1s 后淡出（`ui_resignation_letter.png`）] + 文字爆点 "已发送辞职信！"
↓
进入【摆烂模式】（持续 10 秒，到时自动恢复正常）：
- 武器旋转速度 ×100
- 怪物刷新 / 移动速度 ×5
- 玩家完全无敌
- BGM 切换为癫狂电子摇滚版
- 屏幕轻微抖动持续，模拟混乱
↓
10 秒后【恢复正常】：
- 所有效果回退，玩家恢复可受击
- 摆烂期间的打败和摸鱼值正常累计（咖啡可能攒到上限）
- 继续战斗到 60s 结算
```

### 8.3 摆烂模式设计意图

这是个**纯爽彩蛋 + 隐藏存档**，不是平衡机制：
- 10 秒内玩家无敌 + 转速 ×100 = 屠杀场，看屏幕像放烟花
- 10 秒后回归正常，但靠彩蛋期囤的咖啡和摸鱼值能续命一会儿
- **触发时机决定收益**：早触发（前 30s）= 攒资源多 + 还要再撑 20-30s；晚触发（50s+）= 直接锁胜利
- 撑过 60 秒 + 触发过彩蛋 → 解锁特殊结算文案「老子早就不干了！！！」

### 8.4 触发互动

```
[厕所图标在茶水间闪烁]
玩家走过去 → 弹出小提示「点击进入厕所」
点击 1 次 → 显示 "🚽 1/5"（小气泡）
点击 5 次 → 触发离职信，彻底改变游戏节奏
```

### 11.2 主循环

```js
function tick(dt) {
  if (state !== 'playing') return;
  input.poll();
  player.update(dt);
  weapon.update(dt);          // 旋转、碰撞
  spawner.update(dt, t);
  enemies.forEach(e => e.update(dt));
  collisions.resolve();        // 武器 vs 敌人, 玩家 vs 敌人
  economy.update(killCount);   // 满 20 触发 +1HP
  progression.update(xp);      // 满级触发升级
  easterEgg.update(dt);        // 厕所连点 + 摆烂模式数值修改
  vfx.update(dt);
  taunts.update(dt);
  hud.update();
  render();
  if (player.hp <= 0) gameOver();   // 「今晚加班！！！」
  if (t >= 60) win();                // 「今天按时下班！！」
}
```

## 13. 风险与缓解

| 风险 | 缓解 |
|---|---|
| 1 种武器 + 4 种敌人 → 玩法太单调 | Hook 靠题材反讽 + 摸鱼值正反馈 + 彩蛋话题度撑住；机制深度交给 v2 |
| 厕所彩蛋触发率太低（没人发现）| 茶水间厕所图标做闪烁动画 + 第 3 局起弹"试试连点厕所"提示 |
| 摆烂模式 10s 结束后玩家暴毙 | 10s 期间产出的咖啡 + 摸鱼值给玩家续命缓冲；早触发的玩家有更长缓冲期 |
| 4 种敌人画像不够丰富 | 加大反讽台词池密度（每种 8-10 句） |
| 双回血通道导致玩家死不了 | 摸鱼值需要 20 杀；茶水间 3s 才回 1 HP 且要离开战场，双通道无法同时 cheese |
| 撑满 60s 太容易 / 太难 | 灰度时盯胜率：目标 60-70%，偏离 > 20% 调 SPAWN_RATE |
